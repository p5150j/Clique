## setup

### global 
```
npm i expo -g
npm i firebase-tools -g
npm i expo-cli -g
install xocde 
install android studio

```

### Project
```
cd project
cd frontend
npm i
edit the firebase.initializeApp to add the config details here https://github.com/p5150j/tick-tok-rn/blob/main/frontend/App.js#L21 (ping @p5150j for configs)
expo start
```
